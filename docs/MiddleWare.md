# 中间件说明

## 任务系统 

设计的初衷是统一任务系统逻辑，简化任务系统的开发。   
任务系统分两块内容，数据收集和逻辑计算。   
数据收集在业务开发中，需要开发人员进行埋点收集，由框架进行保存   
任务是否完成的计算在调用任务列表或者详情的时候
